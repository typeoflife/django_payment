# Интеренет магазин с подключенной платежной системой Stripe.

* Цены и товары добавляются через админку.
* При покупке можно указать количество товара.
* Товары можно объединить в один заказ и оплатить одним чеком.

Запуск тестов:
```bash
pytest
```
